Fixing nodes in position and moving them around

There are two more options available when you right click on a node which we need to discuss:



  • Drag: Allows you to move the node to a new position using the mouse
  • Fixed Position: When this option is checked the node is no longer affected by the forces affecting other nodes.
When you fix a node's position a small blue "pin" appears on the sphere's surface:



When you select the drag option you can move the node to a new position by right clicking on the desired location with the mouse.  You can also hold the right button down and drag the node around.  When you release the mouse button, if the node is not fixed, it will spring back into position.  If it is fixed it will stay where you told it to go.

When you move a node in this way it moves to a position at the same depth from the view plane, but directly underneath the mouse pointer.  You can still rotate the whole scene by dragging with the left mouse button so if you want to move the node to a new depth then first rotate the graph so that you get a side on view, and the drag it to the new depth.

When you're finished dragging the node, hit the "Done" button.
